# Services Truck - Product Requirements Document

## 📋 Original Problem Statement
Crear una página web elegante, moderna y muy profesional para un taller especializado en mecánica pesada llamado Services Truck. Diseño dark-theme con negro dominante, naranja para CTAs, transmitiendo potencia, profesionalidad y confianza industrial.

## 🎨 Design System - Final
- **Colors**: 
  - Primary: Negro (#000000) - Dominante
  - Accent: Naranja con gradiente (#FF6B00 → #FF8534) - CTAs
  - Brand: Cyan-Green (#00FFD1) - Detalles
- **Typography**: **Poppins** (Google Fonts) - Moderna y profesional
- **Logo**: Logo real del cliente (camión con grúa) en blanco
- **Buttons**: Border-radius 8px con gradientes
- **Theme**: Dark industrial premium
