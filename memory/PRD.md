# Services Truck - Product Requirements Document

## 📋 Original Problem Statement
Crear una página web elegante, moderna y muy profesional para un taller especializado en mecánica pesada llamado Services Truck. Diseño dark-theme con negro dominante, naranja para CTAs, transmitiendo potencia, profesionalidad y confianza industrial.

## 🏢 Company Information
- **Nombre**: Services Truck
- **Ubicación**: P.º del Limonar, 3, Edificio Neptuno, piso 11D, 04720 Aguadulce, Almería, España
- **Teléfono**: 634 91 92 57
- **WhatsApp**: 634 91 92 57
- **Valoración**: 5.0 estrellas

## 🎯 User Personas
1. **Propietarios de flotas de camiones**: Necesitan mantenimiento regular y reparaciones confiables
2. **Empresas de construcción**: Requieren servicio para excavadoras, grúas y maquinaria pesada
3. **Operadores industriales**: Buscan expertos en motores industriales y algunos motores marinos
4. **Clientes individuales**: Propietarios de camiones que necesitan servicio profesional

## 🎨 Design System (UPDATED)
- **Colors**: 
  - Primary: Negro (#000000) - Dominante
  - Accent: Naranja con gradiente (#FF6B00 → #FF8534) - CTAs y elementos interactivos
  - Brand: Cyan-Green (#00FFD1) - Detalles, iconos y bordes
  - Backgrounds: Gradientes radiales sutiles para profundidad
- **Typography**: Inter font family con jerarquía clara
- **Buttons**: Bordes redondeados (8px) con gradientes y sombras coloridas
- **Cards**: Border-radius 12px con efectos hover dinámicos
- **Theme**: Dark industrial premium con toques de color estratégicos
